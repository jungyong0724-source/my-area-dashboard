1. 프로젝트 개요
1.1 목적
본 문서는 
CESCO
 웹 서비스의 '내 지역 현황판' 기능에 대한 개발 기획서입니다.
해당 기능은 세스코 영업 담당자(지사원)가 자신의 담당 지역 내 고객 현황을 지도 및 리스트 형태로 조회하고,
다양한 필터 조건을 통해 영업 잠재 고객을 탐색하는 데 목적이 있습니다.
1.2 서비스 개요
서비스명
CESCO
 – 내 지역 현황판
서비스 유형
웹 서비스 (모바일 반응형 포함)
주요 사용자
세스코 
현장 직군 
주요 기능
• 담당 지역 고객 현황 지도 기반 조회
• 고객 검색 (텍스트 / 지역 선택)
• 다중 조건 필터링 (업종, 산업분류, 매출 등)
• 고객 상세 정보 및 추천 상품/계약 조회
• 유사 고객 정보 조회

3. 화면 플로우 (User Flow)
3.1 전체 흐름
홈/메뉴
→
내 지역 현황판(지도+리스트뷰)
→
검색 화면(SCR-002)
↕
필터 화면(SCR-003/003B)
→
고객 상세(SCR-004)
3.2 시나리오별 흐름
① 기본 조회 시나리오
내 지역 현황판 진입 → 담당 지역 기준 결과 자동 로드 → 지도뷰/리스트뷰 확인 → 고객 카드 클릭 → 고객 상세 조회
② 키워드 검색 시나리오
검색바 클릭 → 검색 화면(SCR-002) 진입 → 텍스트 입력(고객 상호명 또는 지역) 또는 지역 선택 드롭다운 사용 → 결과 반영된 현황판으로 복귀
③ 필터 적용 시나리오
필터 아이콘(≡) 클릭 → 필터 화면 진입 → 조건 설정(최근 오픈/고객유형/산업분류/매출/추천계약/마이랩 등) → [적용] 버튼 클릭 → 필터 결과 반영된 현황판으로 복귀
④ 빠른 필터칩 시나리오
현황판 화면의 상단 필터칩(최근 오픈 매장 / 2개월 / 3개월 / 고계약률 추정 유사 고객군) 클릭 → 해당 조건 즉시 적용

6. 데이터 정의 및 표시 규칙
6.1 거리 표시
기준점현재 위치으로부터 고객까지의 직선 또는 도로 거리 표시
1km 미만: OOOm 형식, 1km 이상: 0.0km 형식 (소수점 1자리)
6.2 업종 태그
고객 카드 및 상세 화면에 최대 2개 태그 표시
태그 텍스트가 일정 길이 초과 시 말줄임(...) 처리
6.3 검색 결과 카운트
'OOO지사 담당 지역 내 검색 결과 OOO 곳' – 지사명, 건수는 파란색(#1B5EAB) 강조 표시
건수 0건 시 '검색 결과가 없습니다' 안내 문구 표시
6.4 면적
'0,000 평' 형식 (천 단위 콤마)
6.5 전화/팩스번호
'T. 02-1234-5678  F. 02-1234-5679' 형식, 좌우 배치
6.6 유사고객 이용율
'유사한 고객 그룹의 00% 이용중' 형식, 퍼센트는 동적 데이터
